# tp-git--mesa-10-.
sistema embebido para mascotas que combina geolocalización y control de acceso automatizado.
Rastreo y Localización (GPS): El collar incorpora un módulo de posicionamiento global (GPS) que permite obtener las coordenadas geográficas de la mascota en tiempo real.
Control de Acceso selectivo (RFID): Utiliza tecnología de identificación por radiofrecuencia (RFID) para que la mascota pueda abrir una puerta de forma automática al acercarse.
